# Andy & Amy - Engagement Party Invitation

A beautiful one-page engagement party invitation built with React and Vite.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The invitation will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Elegant, romantic design perfect for an engagement party
- Responsive layout that works on all devices
- Beautiful animations and styling
- RSVP functionality via email link
- Single-page application - no routing needed

## Customization

To customize the invitation, edit `src/App.jsx` to update:
- Date and time
- Location details
- RSVP email address
- Any text content
